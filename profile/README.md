# Plexara

[plexara.io](https://plexara.io)

Plexara is an [MCP](https://modelcontextprotocol.io) server that connects AI agents to enterprise data. It sits between your AI clients (Claude, ChatGPT, agents) and your data estate, adding governance, semantic context, and persistent memory.

## What it does

- **Federated SQL access** across PostgreSQL, MySQL, Oracle, SQL Server, Snowflake, BigQuery, MongoDB, Cassandra, Elasticsearch, Kafka, and more.
- **Semantic enrichment** that pulls ownership, glossary terms, quality scores, PII flags, and lineage from your metadata catalog into every response.
- **Knowledge capture** so insights surfaced in AI conversations flow back to the catalog through a review workflow instead of evaporating.
- **Governed by default** with fail-closed authorization, audit logging, and OIDC / API key / OAuth 2.1 auth.
- **Persona-based access** so analysts, executives, and machine-to-machine workflows each get the right tools.
